# Project Setup

## Requirements

- Node.js installed

## Install dependencies

From root folder

```bash
cd backend
npm install

cd ..\frontend
npm install
```

## Run

Open two terminals

**Backend**

```bash
cd backend
npm run dev
```

**Frontend**

```bash
cd frontend
npm run dev
```

## Open

- Frontend: http://localhost:5173
- Backend: http://localhost:3000
